# Matching Game

[![Generic badge](https://img.shields.io/badge/Made%20with-JavaScript-50E040.svg)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

![Game](./readmeimages/play-normal-4.gif)
This is a web game that requires you to guess and match cards. The less moves you take the higher score you get. There's an Expert mode for someone who wants more challenges.
## Parts of the Game
* Deck
<img src="./readmeimages/deck.png" width="300px" height="100%" alt="deck">
* Score Panel

## How to Play
1. Click on a card to open it
2. Match all cards to win the game.
## Planned Changes
* A new difficulty that will be more challenging: if no cards are matched in 3 continuous moves, a pair of matched cards will close and switch positions with 2 other cards.
* Add theme selection: different sets of images for cards.
* Add sound effects.
